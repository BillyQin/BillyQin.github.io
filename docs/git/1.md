title: git常用命令
---
### 1. 初始化
``` sh
# 为当前目录创建git仓库
git init

# 将指定文件添加到暂存区
git add <filename>

# 将暂存区文件提交到仓库
git commit -m <message>

# 将本地仓库代码提交到远程仓库
git push
```
### 2. 标签管理
切换到需要打标签的分支上

#### 增加标签
> 创建的标签都只存储在本地，不会自动推送到远程。
``` sh
# 增加标签
git tag <tagname>

# 为某一次commit打标签
git tag <tagname> <commitId>

# 指定标签详细信息
git tag -a <tagname> -m <message>
```
#### 删除标签
``` sh
# 删除本地标签
git tag -d <tagname>

# 删除远程标签
git push origin :refs/tags/<tagname>
```

#### 查看标签
``` sh
# 查看本地所有标签
git tag

# 查看标签详细信息(commit、author、date)
git show <tagname>
```
#### 推送标签

``` sh
# 推送单个标签
git push origin <tagname>

# 推送全部本地标签
git push origin --tags
```
